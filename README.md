## 🎯 Project Overview
This dashboard answers: **"Where in the world can I earn well AND live comfortably?"**

It analyzes salary data vs cost of living across **47 global cities** spanning **8 regions**.

## 📊 Dashboard Features
- 🗺️ Interactive World Map with 4 selectable metrics
- 📊 Salary vs Cost grouped bar charts
- 🏆 Top 10 most affordable & highest salary cities
- 🔍 City Deep Dive with radar chart comparison
- 🎛️ Filters by region, salary range, city count

## 🛠️ Tech Stack
- Python · Streamlit · Plotly · Pandas

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run app/dashboard.py
```
